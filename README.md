# Gestor de videojuegos

Proyecto básico para creación de listado de videojuegos, ya sea completados, comprados, etc.
Usa **Node y SQLite** en el backend y **Vue (a través de Vite)** en el frontend.

## Instrucciones de uso

### Backend

Desde el terminal, navega hasta la carpeta del backend:
```bash
cd backend
```

Instala las dependencias:

```bash
npm install
```

Arranca el server:

```bash
npm run dev
```

Se abre el servidor en http://localhost:3000 y se crea la base de datos videogames.db

### Frontend

Se repiten los mismos pasos solo que desde la carpeta de frontend:

```bash
cd frontend
```

El frontend se abrirá en http://localhost:5173

Hay un par de colecciones dentro de la carpeta backend que pueden ser importadas a Hoppscotch o Postman para probar las funciones de los endpoints de la API.
